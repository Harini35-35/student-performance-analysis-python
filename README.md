# 📊 Student Performance Analysis using Python
# Internship Project

## 📌 Project Overview

This project was completed as part of my Data Science/Data Analysis with Python Internship at Sumerix Global.

- Data Cleaning
- Exploratory Data Analysis
- Statistical Analysis
- Data Visualization
- Conclusions
